# 🌾Analyse de la performance agricole et financière 50 planteurs – Sénégal (2019–2023)

## 📌 Contexte
Analyse de la performance agricole et financière de 50 planteurs
répartis dans 5 zones agroécologiques du Sénégal, dans le cadre
d'un programme de microcrédit agricole d'une ONG.

## 📊 Aperçu du Dashboard
![Vue globale](screenshots/01_vue_globale.png)

## 🎯 Objectifs
- Identifier les cultures les plus productives par zone
- Mesurer l'impact du microcrédit sur les rendements
- Analyser les comportements de remboursement
- Cartographier les performances géographiques

## 🗂️ Structure des données
| Table | Description |
|---|---|
| DimRegion | 5 zones agroécologiques |
| DimVille | Villes par région |
| DimCulture | Cultures (riz, arachide, anacarde...) |
| DimFermier | 50 planteurs localisés |
| FaitAgriculture | Mesures annuelles 2019–2023 |

## 🛠️ Outils utilisés
- Power BI Desktop
- CSV / Excel

## 📈 KPIs principaux
- Productivité (rendement/km²)
- Taux de remboursement
- Taux d'adoption du microcrédit
- Évolution annuelle des rendements
